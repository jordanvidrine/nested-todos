Nested Todos App

Info

This app was created as an exercise for the Watch and Code premium course by Gordon Zhu

To add a todo, type your text into the text input and click on the add button.

To add a Sub Todo, type in you text as before, but click the 'add sub-todo' button of the task you wish to add on to.

Clicking on the square box next to a Todo marks it as complete. A todo with uncomplete subtodos cannot be checked.

Double Click to edit todos
